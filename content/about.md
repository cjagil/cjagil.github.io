I'm Curtis. This site is a commonplace book turned inside out — a place to store
thoughts, work them over in essays, and keep the things I read where I can find
them again.

The site is deliberately plain: hand-written text, a serif font, almost no
JavaScript, and files that should still open in a browser decades from now. It's
built with [Hugo](https://gohugo.io) and served as static HTML, which means it
costs essentially nothing to run.

You can reach me at [curtis.j.gile@gmail.com](mailto:curtis.j.gile@gmail.com).

### Colophon

Written in Markdown, compiled to static HTML, and hosted for free. No database,
no tracking, no cookies. If something looks broken, it's probably because I'm
still tinkering.
